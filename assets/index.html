Portfolio

Covid Data Analysis Using SQL

In this COVID data analysis project, I explored various aspects of the pandemic using SQL. Highlights include examining the relationship between total cases and deaths, identifying countries with the highest infection rates and death counts, and analyzing vaccination progress globally. Insights gained include understanding the impact of population size on infection and vaccination rates, as well as tracking trends in COVID-related metrics across continents and countries.

Machine Learning Projects

Here I showcase some of my machine learning and Data Science projects for model prediction, classification and regression.

End to End Data Science Project

This end-to-end data science project encompasses the entire lifecycle, from data preprocessing and feature engineering to model training, testing, and deployment. Leveraging robust pipelines, advanced preprocessing techniques, and thorough feature engineering, it optimizes model performance for accurate predictions. Customer Churn Prediction

This customer churn prediction project utilizes logistic regression, random forest, and gradient boosting classifiers to accurately predict customer churn. Achievements include evaluating model performance metrics such as accuracy, precision, recall, F1 score, and ROC AUC, enabling informed decision-making for customer retention strategies. Additionally, the project incorporates visualizations like ROC curves to provide a comprehensive analysis of model performance across different algorithms.

Spam Email Detection

In this spam email detection project, a Naive Bayes algorithm achieved an impressive accuracy of 96.68%. The confusion matrix reveals strong performance with only 37 false negatives and no false positives, indicating robust spam identification capabilities. The classification report highlights precision, recall, and F1-score metrics, demonstrating the model's ability to accurately classify spam and non-spam emails. This project contributes to enhancing email security by effectively filtering out unwanted spam content, thus improving user experience and overall system efficiency.
